# Описание
CLI утилита для обработки PNG изображений

## Пример использования

```bash
cargo run \
  --bin image_processor \
  -- --input <INPUT_PATH> \
  --output <OUTPUT_PATH> \
  --params <PARAMS_PATH> -- \
  plugin <PLUGIN>.dylib
```
